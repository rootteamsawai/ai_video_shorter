# サイドバーメニュー 実装プラン

## ファイル構成

```
src/
  app/
    layout.tsx                    # 変更: サイドバーレイアウトを適用
    history/
      page.tsx                    # 新規: 履歴ページ
  components/
    sidebar.tsx                   # 新規: サイドバーコンポーネント
  lib/
    job-store.ts                  # 変更: 全ジョブ取得関数を追加
  app/api/
    jobs/
      route.ts                    # 新規: ジョブ一覧API
```

## 実装順序

### Step 1: ジョブ一覧取得機能（バックエンド）

**対象ファイル:**
- `src/lib/job-store.ts`
- `src/app/api/jobs/route.ts`

**やること:**
1. `job-store.ts` に `getAllJobs()` 関数を追加
   - `storage/jobs/` ディレクトリ内の全ジョブを取得
   - 作成日時の降順でソート
2. `GET /api/jobs` エンドポイントを作成
   - `getAllJobs()` を呼び出してジョブ一覧を返す

### Step 2: サイドバーコンポーネント

**対象ファイル:**
- `src/components/sidebar.tsx`

**やること:**
1. サイドバーコンポーネントを作成
   - PC: 左サイドバー固定表示（幅 240px）
   - モバイル: ハンバーガーメニューで開閉
2. メニュー項目
   - ホーム（`/`）
   - 履歴（`/history`）
3. 現在のパスに応じてアクティブ状態を表示

### Step 3: レイアウト変更

**対象ファイル:**
- `src/app/layout.tsx`

**やること:**
1. サイドバーを組み込んだレイアウトに変更
2. PC: サイドバー + メインコンテンツの2カラムレイアウト
3. モバイル: サイドバーをオーバーレイ表示

### Step 4: 履歴ページ

**対象ファイル:**
- `src/app/history/page.tsx`

**やること:**
1. ジョブ一覧APIを呼び出して表示
2. 各ジョブの表示項目
   - ステータス（アイコン or バッジ）
   - 作成日時
   - ジョブ詳細へのリンク
3. ジョブがない場合の空状態表示
