# 目的

`.claude/incidents.md` の未昇格エントリを棚卸しし、
共通パターンを抽象ルールとして `.claude/rules/` に昇格する

# 手順

1. `.claude/incidents.md` を読む
2. 全エントリの「教訓」欄を一覧し、共通パターンを探す
3. 以下のいずれかに該当するものを昇格候補とする：
   - 同じカテゴリの問題が3件以上
   - 同じ教訓が2回以上
   - 明らかに今後も頻出する問題（即時昇格）
4. 昇格候補について、複数の具体的問題を包含する汎用ルールを1〜2文で言語化する
5. **ユーザーに昇格候補を提示し、承認を得る**
6. 承認後、`.claude/rules/` 内の適切な既存ルールファイルに数行追記する（新規ファイルは作らず既存に足す）
7. 昇格元エントリの見出しに `[昇格済み]` を付ける（`incidents.md` にそのまま残す。別ファイルへの分離は件数が増えてからでよい）
8. 昇格候補がない場合はその旨を報告して終了する
